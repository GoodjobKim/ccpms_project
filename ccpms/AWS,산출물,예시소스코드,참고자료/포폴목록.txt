음식점리스트(KEEP)
client : 	shop	현재 위치한 지역의 음식점 리스트
		맛집, 방송에 출연적이 있는 음식점 을 선택할수 있는 카테고리 기능
	login	회원가입기능, 로그인, 로그아웃, 회원탈퇴
		 위시리스트(장바구니) 에 추가 할수 있는 기능
	myPage	내정보 수정, 내 위시리스트 등을 볼수 있는 MyPage기능
	contact	고객센터 게시판(자주묻는 질문) 1:1 contact할수 있는 기능

admin : 	login	관리자계정 회원가입 로그인 기능
	goods	음식점리스트 등록, 삭제, 수정, 조회 할수 있는 기능
	manage	유저 관리 기능(언제 몇명이 회원가입했는지 등등)
	contact	고객센터 게시판에 1:1contact에 댓글을 달 수 있는 기능


현재 진행중인 공연 목록
client:	goods	현재 진행중인 공연 리스트, 상품상세정보
		뮤지컬, 연극,콘서트, 클래식, 가족/아동, 뮤직페스티벌, 공연제 등의 대주제
		장르, 지역, 공연기간, 날짜, 공연중/공연예정 등의 소주제
		member	회원가입기능, 로그인, 로그아웃, 회원탈퇴
		order	장바구니(cart)/예약(order)등을 할 수 있고		//장바구니는 위시리스트(찜목록)같은 개념
		myPage	내정보 수정, 장바구니(cart)/예약(order) 내역을 볼수있게, 회원탈퇴
		contact	고객센터 게시판(자주묻는 질문) 1:1 contact할수 있는 기능
		↑
		common	가장 메인화면- 새로 추가된 공연/정보, 인기있는 공연/정보, 공통기능
		↓
admin	goods	공연 리스트 추가, 수정, 삭제, 조회
		member	유저 관리 기능, (회원가입을 몇명이 했는지), 관리자 계정 회원가입, 로그인, 로그아웃
		contact	고객센터 게시판에 자주묻는질문 추가, 1:1contact온거 댓글
		order	현재 예약된 공연을보여주는 기능 (시장조사, 어떤 공연을 좋아하는지 등등 파악하기위해)
