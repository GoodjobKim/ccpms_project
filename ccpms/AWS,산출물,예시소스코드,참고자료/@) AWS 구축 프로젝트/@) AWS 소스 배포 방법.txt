@) AWS 소스 배포 방법 [ 배포 소스 예시 기준 ]

1) 이클립스에서 

	1) jdbc.properties 파일에서 DB접속 정보변경			
	2) 파일저장경로 및 로그파일 저장 경로 변경 ->	CommonController, AdminGoodsController에 있는 GOODS_IMAGE_REPO_PATH


2) 이클립스에서 export > war파일 생성


3) putty로 원격 접속하여 홈디렉터리로 이동

	$ cd /var/lib/tomcat9/webapps            // 소스가 저장되어 있는 파일로 이동
	$ sudo rm -rf (프로젝트명)*		// 기존 파일들 삭제


4) 파일질라로 접속하여

	/var/lib/tomcat9/webapps위치에 (프로젝트명).war 파일 업로드


5) WAS 재시작

	$ sudo service tomcat9 restart

	
DB에 data Export할때 전체 스키마를 export하지 말고 각각의 테이블에서 Export해서 aws에 해당 table에 import하기

***Export시 csv가 아니라 json으로 해야함!!
***Import시 새로 만들지 말고 기존껄로 하고 Goods같은경우는 file_repo에 있는 사진들을 aws의 file_repository에 수작업으로 옮겨줘야함!!!!
